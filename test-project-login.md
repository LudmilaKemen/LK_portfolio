🧪 Login Page Testing Project
📌 О проекте

Этот проект демонстрирует процесс ручного тестирования формы авторизации на тренировочном сайте.
Цель — проверить корректность работы логики входа, обработку ошибок и UX-поведение системы.

🌐 Тестируемый ресурс

👉 https://practicetestautomation.com/practice-test-login/

🎯 Цель тестирования

Проверить:

корректную авторизацию пользователя
обработку валидных и невалидных данных
поведение системы при ошибках ввода
качество сообщений об ошибках (UX)

🐞 Найденные дефекты
🔴 Bug 1: Некорректная работа авторизации с валидными данными

Описание:
При вводе корректных учетных данных система не выполняет успешный вход.

Шаги воспроизведения:

Открыть страницу логина
Ввести username: student
Ввести password: Password123
Нажать кнопку Login

Фактический результат:
Отображается сообщение об ошибке (пароль недействителен)

Ожидаемый результат:
Пользователь успешно авторизуется и переходит в систему

Severity: Critical
Priority: High

🟠 Bug 2: Несогласованные сообщения об ошибках

Описание:
При разных сценариях неверного ввода система отображает разные тексты ошибок.

Фактический результат:

"пароль недействителен"
"логин или пароль неверный"

Ожидаемый результат:
Единое и понятное сообщение об ошибке

Severity: Minor
Priority: Medium
